Fluid 加入 Valine 评论模块

本文默认你已经安装好nodejs、hexo等环境，并且已经根据 Fluid指南 安装好网站主题！

博主之前用过 utterances 评论模块，但是如果不换个姿势冲浪这东西就很难刷出来，等于没用，所以转向使用 Valine 了

1. 关于Valine

Valine 是一款基于LeanCloud的快速、简洁且高效的无后端评论系统。
理论上支持但不限于静态博客，目前支持Hexo、Jekyll、Typecho、Hugo、Ghost 等博客程序。

2. Valine 获取 APP ID 和 APP Key

① 打开Valine 官方网站，点击右侧快速开始，根据提示注册并登陆 Valine

② 创建应用

③ 点击右上角小齿轮

④ 如图下图获取 AppID 和 Appkey

3. 添加 AppID 和 AppKey

打开 网站根目录/source/data/fluid_config.yml 修改第211-213行

comments:  # 评论
enable: true  # 开启评论
type: valine  # 指定使用的评论模块  available: disqus | valine | gitalk | utterances | changyan | livere  推荐 utterances

然后至第267-268行，填写 AppId 和 AppKey

valine:
  appid: # 从 LeanCloud 的应用中得到的 appId
  appkey: # 从 LeanCloud 的应用中得到的 APPKey
  placeholder: 说点什么 # 评论框占位提示符
  path: window.location.pathname # 当前文章页路径，用于区分不同的文章页，以保证正确读取该文章页下的评论列表
  avatar: retro # Gravatar 头像展示方式
  meta: ['nick', 'mail', 'link']  # 评论者相关属性
  pageSize: 10 # 评论列表分页，每页条数
  lang: zh-CN # zh-CN | zh-TW | en | ja
  highlight: true # 代码高亮
  recordIP: true # 是否记录评论者IP
  serverURLs: # 该配置适用于国内自定义域名用户, 海外版本会自动检测(无需手动填写)
  mathJax: false # 是否启用MathJax

至此已经配置完毕，使用hexo clean && hexo g && hexo s命令，然后在本地打开 localhost:4000 即可见到评论模块已经出现！
